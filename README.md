# EFramework

[![License](https://img.shields.io/badge/license-MIT-blue.svg?label=License)](LICENSE)
[![Discord](https://img.shields.io/discord/1422114598835851286?label=Discord&logo=discord)](https://discord.gg/XMPx2wXSz3)

开箱即用的框架解决方案，支持跨平台环境和多业务场景，让开发变得更简单。

## 功能特性

- [EFramework for DotNet](https://github.com/eframework-io/DotNet)：基于 .NET 平台的开发框架，提供了一系列高效且易用的类库，专注于快速开发稳定可靠的应用。
- [EFramework for Go](https://github.com/eframework-io/Go)：基于 Go 语言的开发框架，提供了一系列高效且易用的类库，专注于快速开发稳定可靠的应用。
- [EFramework for Node](https://github.com/eframework-io/Node)：面向 Node.js 环境的开发框架，提供了一系列高效且易用的类库，专注于快速开发稳定可靠的应用。
- [EFramework for Unite](https://github.com/eframework-io/Unite)：面向 Unite 平台的开发框架，提供了一系列高效且易用的类库，专注于快速开发稳定可靠的应用。
- [EFramework for Unity](https://github.com/eframework-io/Unity)：基于 Unity 引擎的开发框架，提供了一系列高效且易用的类库，专注于快速开发稳定可靠的应用。
- [EFramework for VSCode](https://github.com/eframework-io/VSCode)：面向 VSCode 编辑器的拓展工具集，提供了一系列高效且易用的插件，专注于简化开发流程、提高迭代速度。
